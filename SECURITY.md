# Security Policy

AgentShield is a security-focused project. If you discover a security vulnerability,
please report it privately so it can be handled before public disclosure.

## Reporting a vulnerability

- Prefer GitHub Security Advisories:
  <https://github.com/limaronaldo/agentshield/security/advisories/new>
- If the issue is active and contains sensitive details, do not create a public issue.

Please include:

- affected component and version (or commit SHA)
- attack scenario and impact
- reproduction steps and payload examples
- files, logs, or commands that demonstrate the issue

## What to expect

- We aim to acknowledge reports quickly and will keep you informed of triage and fix status.
- We may request additional details or a proof-of-concept patch.
- Once fixed, we will publish disclosure details in release notes when appropriate.
