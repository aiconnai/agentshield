# Contributor Covenant Code of Conduct

## Our Pledge

We as members, contributors, and maintainers of this project pledge to make
participation in our community a harassment-free experience for everyone,
evening out power differences of ability, identity, and experience.

## Our Standards

Examples of behavior that helps our community:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Giving and accepting constructive feedback gracefully
- Taking responsibility and apologizing to those affected by mistakes

Examples of unacceptable behavior:

- Harassment of any kind
- Trolling, personal attacks, and insulting comments
- Public or private abuse through email or other channels
- Publishing others' private information without explicit consent
- Any conduct inappropriate for a professional community

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported by opening a GitHub issue with a clear, concise description and contact
information. Maintainers will review and take appropriate action, including warning,
content removal, temporary bans, or permanent exclusion from contribution spaces.

This Code of Conduct applies to project code repositories, issue trackers,
code review, and community spaces where this project is represented.

## Scope

This policy applies to all project venues, including GitHub discussions, pull
requests, and any related communication channels.
