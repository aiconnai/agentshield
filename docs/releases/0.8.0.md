# AgentShield 0.8.0 Release Notes

Release date: 2026-06-05

## Scope

AgentShield 0.8.0 updates the project from the early MCP/OpenClaw-centered documentation to the current multi-framework scanner scope. The release line covers broader adapter support, an expanded built-in rule surface, suppression and baseline workflows, DSSE scan attestations, and optional runtime egress enforcement.

## Notable changes since 0.2.4

- Added supported adapters for GPT Actions and Cursor Rules alongside MCP, OpenClaw, CrewAI, and LangChain/LangGraph.
- Expanded built-in detection coverage beyond the original early rule set, including additional checks for metadata SSRF, download-and-execute behavior, overbroad filesystem access, unsafe deserialization, archive traversal, and secret leakage.
- Added suppression management through `agentshield suppress` and `agentshield list-suppressions`.
- Added baseline workflows through `agentshield scan --baseline` and `--write-baseline`.
- Added egress policy generation through `agentshield scan --emit-egress-policy`.
- Added DSSE scan result certification through `agentshield certify`.
- Added feature-gated runtime egress enforcement through `agentshield wrap` when built with the `runtime` or `full` feature.
- Refreshed public README and GitHub Action metadata to describe general supported adapters instead of MCP/OpenClaw-only language.
