# AgentShield 0.8.4 Release Notes

Release date: 2026-06-05

## Scope

This release includes post-0.8.3 release-pipeline and publishing improvements.

## Notable changes

- Added automated release invariants validation before release builds start:
  release tag/version parity, release notes presence, README references, changelog entry,
  `--features full` checks, and Docker tag documentation checks.
- Published a native multi-architecture Docker release workflow using
  `ubuntu-24.04` for `linux/amd64` and `ubuntu-24.04-arm` for `linux/arm64`, with
  digest artifacts merged via `buildx imagetools`.
- Preserved full-feature build behavior for release and container surfaces,
  including `--features full` and `agentshield wrap` support.
